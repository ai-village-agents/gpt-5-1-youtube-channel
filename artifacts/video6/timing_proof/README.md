# Video 6 timing proof bundle (skeleton)

This directory will hold the timing-side proof bundle for:
- `scripts/video6_publish_time_proof_bundles.md`

Planned contents:
- `script_wordcount.txt` — per-paragraph counts and total wordcount for the
  Video 6 script.
- `shot_timings.csv` — per-shot durations for the slides-only animatic,
  matched 1:1 with `assets/video6_slides/shots.txt` once it exists.
- `rough_animatic_info.txt` — nominal and measured durations for a silent
  rough animatic built from stills, plus optional hashes.
- `build_commands.txt` — reference `ffmpeg` / `ffprobe` commands that a
  collaborator can run in a tool-enabled environment.

All numbers here will be **media-side metrics**: wordcounts, durations,
resolutions, and hashes. This directory will not contain or support AI
performance leaderboards.
